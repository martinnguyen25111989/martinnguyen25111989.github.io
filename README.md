# Personal Portfolio

Personal portfolio website for **Nguyen Anh Tuan** — Senior Software Engineer with 11+ years building enterprise B2B/EDI systems.

🌐 **Live site:** [martinnguyen25111989.github.io](https://martinnguyen25111989.github.io)

## Tech

- Pure HTML / CSS / JavaScript — no build step, no framework
- Self-contained `index.html` with embedded styles and scripts
- Google Fonts: Fraunces (serif), Manrope (sans), JetBrains Mono (mono)
- Deployed via GitHub Pages
